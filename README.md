# Конструктор маршрутів ПДР

Статичний сайт для GitHub Pages. Працює без бекенду: усі маршрути, дороги, вузлики, знаки, примітки, стрілки й елементи зберігаються у `localStorage` браузера.

## Файли

- `index.html`
- `style.css`
- `app.js`

## Запуск локально

```bash
cd route_map_builder_v9
python -m http.server 8000
```

Потім відкрити:

```text
http://localhost:8000
```

## GitHub Pages

1. Завантажити файли в репозиторій.
2. Увімкнути GitHub Pages для гілки `main` або `master`.
3. Відкрити посилання GitHub Pages.

## Основна логіка

- Вкладка `Інструменти`: малювання доріг, переміщення, видалення, стрілки-виноски, примітки, зебра, лежачий поліцейський, світлофор.
- Вкладка `Знаки ПДР`: повний перелік знаків з pdr.infotech.gov.ua/theory/road-signs, згрупований за категоріями.
- Вкладка `Маршрути`: створення, перейменування, видалення та переміщення маршрутів.
- Вкладка `Дані`: експорт/імпорт JSON і очищення активного маршруту.

## Масштабування елементів

Знаки, зебра, лежачий поліцейський і світлофор зберігають розмір у метрах на карті. Тому при віддаленні карти вони стають меншими на екрані, а при наближенні — більшими.


## v9
- Градієнт дороги рахується від фактичної сумарної довжини маршруту, а не від кількості вузликів.
- Примітки та стрілки-виноски масштабуються разом із картою.
- Вузлики доріг показуються лише для інструментів «Обрати / перемістити» та «Малювання доріг».


## v9
- Примітки більше не стискають текст вертикально при віддаленні карти.
- На малому масштабі примітка автоматично переходить у компактний бейдж з 1–2 літерами або коротким скороченням.
